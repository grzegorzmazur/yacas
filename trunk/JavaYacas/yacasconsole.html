Note: to allow supporting copy-pasting from and to this applet, you need to enable access to the clip board on your computer. On Unix-style computers this can be done by appending the following lines to the file ~/.java.policy (create the file if it does not exist yet):

grant codeBase "http://www.xs4all.nl/~apinkus/*" { permission java.awt.AWTPermission "accessClipboard"; };

You can then copy with CTRL-c and paste with CTRL-v.
