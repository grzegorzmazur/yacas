: , @ , /@ , .. .

Functional operators

:

Standard math library
Calling Sequence:
item : list
string1 : string2
Parameters:
item - an item to be prepended to a list
list - a list
string1 - a string
string2 - a string
Description:
Prepends an item to a list, or a string.
Examples: 
In> a:b:c:{}
Out> {a,b,c};
In> "This":"Is":"A":"String"
Out> "ThisIsAString";
See Also:
Concat , ConcatStrings .

@

Standard math library
Calling Sequence:
function @ arguments
Parameters:
function - a string representing a function
arguments - a single arguments, or a list of arguments
Description:
Applies an operator to arguments. Equivalent to Apply
Examples: 
In> "Sin" @ a
Out> Sin(a);
In> {{a},Sin(a)} @ a
Out> Sin(a);
In> "f" @ {a,b}
Out> f(a,b);
See Also:
Apply .

/@

Standard math library
Calling Sequence:
function /@ arguments
Parameters:
function - a string representing a function
arguments - a list of arguments
Description:
Applies an operator to arguments in a list, in a threaded manner.
Examples: 
In> "Sin" /@ {a,b}
Out> {Sin(a),Sin(b)};
In> {{a},Sin(a)*a} /@ {a,b}
Out> {Sin(a)*a,Sin(b)*b};
See Also:
MapSingle .

..

Standard math library
Calling Sequence:
n .. m
Parameters:
n - an integer
m - an integer
Description:
Returns a list of numbers from n upto m.
Examples: 
In> 1 .. 4
Out> {1,2,3,4};
