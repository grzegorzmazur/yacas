Polynomials

Expand

Standard math library
Calling Sequence:
Expand(expr)
Expand(expr,var) Expand(expr,varlist)
Parameters:
expr - a univariate polynomial expression
var - a variable
varlist - a list of variables
Description:
Expand(expr) : expands a univariate. Example: Expand((1+x)^2) would evaluate to 1+2*x+x^2. If the expression depends on more than one variable, you can specify which variable to expand to using Expand(expr,var). Also, you can expand to multiple variables, by specifying the order in which to expand, in a list, using Expand(expr,{varlist}).
Examples: 
In> Expand((1+x)^4,x)
Out> x^4+4*x^3+6*x^2+4*x+1;

Coefficients of univariate polynomials

Standard math library
Calling Sequence:
Degree(expr)
Degree(expr, var)
Coef(expr,var,order)
Parameters:
expr - an expression
var - a variable
order - an integer
Description:
Degree returns the degree of a polynomial. Example: Degree((1+x)^2); evaluates to 2.

The version accepting an additional variable as an argument can be used to get the degree of a multivariate polynomial with respect to that variable. Example: Degree(a+b*x^3,a);returns 1.

Coef returns the coefficient of order for expression expr treated as a univariate with respect to the variable var.

The argument to parameter order can also be a list of integers, in which case this function returns a list of coefficients.

Examples: 
In> e := Expand((a+x)^4,x)
Out> x^4+4*a*x^3+(a^2+(2*a)^2+a^2)*x^2+(a^2*2*a+2*a^3)*x+a^4;
In> Degree(e,a)
Out> 4;
In> Coef(e,a,2)
Out> 6*x^2;
In> Coef(e,a,0 .. 4)
Out> {x^4,4*x^3,6*x^2,4*x,1};

Content and primitive parts of univariate polynomials

Standard math library
Calling Sequence:
Content(poly)
PrimitivePart(poly)
Parameters:
poly - a univariate polynomial with constant coefficients
Description:
Content determines the content of a univariate polynomial. The content is the greatest common divisor of each term in the polynomial. The content of 2*x^2+4*x should be 2*x for instance.

PrimitivePart determines the primitive part of a univariate polynomial. This is defined as PrimitivePart(poly)*Content(poly) = poly, and can easily be checked with Expand(PrimitivePart(poly)*Content(poly)) which should be equal to Expand(poly).

Examples: 
In> e := 2*x^2+4*x
Out> 2*x^2+4*x;
In> Content(e)
Out> 2*x;
In> PrimitivePart(e)
Out> x+2;
In> Expand(Content(e)*PrimitivePart(e))
Out> 2*x^2+4*x;

LeadingCoef and Monic

Standard math library
Calling Sequence:
LeadingCoef(poly)
LeadingCoef(poly,var)
Monic(poly)
Monic(poly,var)
Parameters:
poly - a univariate polynomial
Description:
LeadingCoef returns the leading coefficient of the polynomial poly. If there are more variables in poly, you can specify which variable is the main one, by adding it as an argument. Monic returns the monic part of the polynomial poly. This is poly/LeadingCoef(poly). This function also accepts a second argument, specifying the variable of the univariate polynomial. The leading coefficient of the monic polynomial will thus be one.
Examples: 
In> e := a*x^2 + b
Out> a*x^2+b;
In> LeadingCoef(e,x)
Out> a;
In> Monic(e,x)
Out> x^2+b/a;

RandomPoly

Standard math library
Calling Sequence:
RandomPoly(var,deg,coefmin,coefmax)
Parameters:
var - free variable for resulting univariate polynomial
deg - degree of resulting univariate polynomial
coefmin - minimum value for coefficients
coefmax - maximum value for coefficients
Description:
RandomPoly generates a random polynomial in variable var, of degree deg, with coefficients ranging from coefmin to coefmax (inclusive).
Examples: 
In> RandomPoly(x,3,-10,10)
Out> 3*x^3+10*x^2+ -4*x-6;
In> RandomPoly(x,3,-10,10)
Out> -2*x^3+ -8*x^2+8;

Horner

Standard math library
Calling Sequence:
Horner(expression,variable)
Parameters:
expression - an expression variable - a variable
Description:
Horner turns a univariate polynomial into a Horner form, ((x+a)*x+b)*x+c ... . This is usually an efficient way to evaluate a polynomial.
Examples: 
In> expr1:=Expand((1+x)^4) 
Out> x^4+4*x^3+6*x^2+4*x+1 
In>  Horner(expr1,x) 
Out> (((x+4)*x+6)*x+4)*x+1

ExpandBrackets

Standard math library
Calling Sequence:
ExpandBrackets(expr)
Parameters:
expr - a multivariate polynomial
Description:
ExpandBrackets goes further than Expand, in that it expands all terms.
Examples: 
In> Expand((a-x)*(b-x),x)
Out> x^2-(b+a)*x+a*b;
In> Expand((a-x)*(b-x),{x,a,b})
Out> x^2-(b+a)*x+b*a;
In> ExpandBrackets((a-x)*(b-x))
Out> a*b-x*b+x^2-a*x;
