Fun with derivatives part 1: defining the concept of a derivative in Yacas

You can start using the code developed in the accompanying article by calling the function dd. Here is a list with examples:

  • dd(x,a*x^n)
  • dd(x,Sin(a*x))
  • dd(x,1-a*x^Sqrt(2))
  • dd(x,Exp(1+Sin(a*x)))
etcetera.

You can verify that the above examples return the same result as the function D built in to Yacas. For example, dd(x,Exp(1+Sin(a*x))) and D(x) Exp(1+Sin(a*x)) should give the same results.

You can extend the code yourself to handle other functions not covered yet, by going to this page.

You could for example define the derivative of Sqrt(x) with
dd(_x,Sqrt(_y)) <-- dd(x,y)/Sqrt(y);.

Alternatively you can restart the console (so it forgets the previous transformation rule for taking the derivative of a square root), and define the derivative of the square root as
dd(_x,Sqrt(_y)) <-- (1/2)*y^(1/2-1)*dd(x,y); See the difference the two methods make when repeatedly taking derivatives:
dd(x,dd(x,dd(x,dd(x,dd(x,Sqrt(x)))))).
